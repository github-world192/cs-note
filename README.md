這堂課會先教你在你的windows 機器上面安裝virtualbox虛擬機
，接著會在虛擬機上面安裝linux其中一個發行版[(ubuntu)](https://releases.ubuntu.com/jammy/)，
並使用內建[gcc編譯器](/) 與[vim編輯器](/) 撰寫與編譯c語言程式碼 。
